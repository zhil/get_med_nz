# Get Med NZ

Get Med NZ is a [Daml](https://daml.com/) application for a cohesive, personal medications system.

## Installation

```
# Clone the Github repository
$ git clone https://github.com/quid-agis/get_med_nz
$ cd get_med_nz

# Build the Daml application
$ daml build
```
